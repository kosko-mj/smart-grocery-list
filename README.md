# Smart Grocery List

A grocery shopping app that lets you create reusable shopping lists and dynamically reorder items based on which store you're shopping at.

## Features

- Create and save multiple shopping lists
- Add items with quantities to any list
- Store-specific aisle mappings (items reorder based on selected store)
- Persistent data storage with SQLite
- REST API built with Express

## Tech Stack

- **Backend**: Node.js, Express
- **Database**: SQLite
- **Frontend**: HTML, CSS, vanilla JavaScript
- **API**: RESTful endpoints

## Purpose

This project was built to deepen understanding of backend development, database design, and client-server communication without relying on AI-generated code.
